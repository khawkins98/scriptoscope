# slimes 1.5

**Author:** JUN (j-habu@air.linkclub.or.jp)
**Year:** 1998
**License:** No explicit license terms in the archive ReadMe ("This is my first scheme and I am expecting to create many more"). Distributed via the period Kaleidoscope community as freeware by convention.

## About

slimes 1.5 is a 1998 Kaleidoscope scheme by JUN. The bundle ships desktop patterns, a custom cursor, and a set of stuffit-icon overlays; this port covers the window-chrome scheme.

*"This is my first scheme and I am expecting to create many more in the not too distant future. I hope you like it! ... Slimes ver.1.5, 18/Aug/1998 updated. Desktop patterns, Cursor, and some zip icons included."* — from the bundle's ReadMe.


## Sourcing

This bundle was ported from a `.sit` archive distributed via the **Mac Themes Garden** community archive:

- **Mac Themes Garden:** [macthemes.garden](https://macthemes.garden/) — a community-maintained gallery + archive of Kaleidoscope schemes from the 1997–2003 era, with reference renders and links to the original `.sit` downloads.
- **Source `.sit`:** `slimes.sit` (StuffIt), resource fork extracted to `scheme.rsrc` via Scriptoscope's in-browser StuffIt decoder ([`tools/sit-wasm/`](../../tools/sit-wasm/)).

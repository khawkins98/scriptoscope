# BeOS R5.0.3

**Author:** Jon Alexander (jon1101@chartermi.net)
**Year:** 2002
**License:** Freeware Kaleidoscope scheme. BeOS r5.0.3 ©2002 Jon Alexander; distributed freely via the Kaleidoscope community.

## About

From the scheme's Readme: *"BeOS r5.0.3 is a Kaleidoscope scheme based on the BeOS. This scheme requires Kaleidoscope 2.3 or higher... The most notable change is that there are no collapse buttons on any windows. The BeOS didn't have them, so neither does the scheme."* The Be Operating System was made by Be Inc. before its intellectual property was acquired by Palm; version 5.0.3 was the last release.

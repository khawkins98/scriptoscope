# Windows 95

**Author:** Scott Naness (asimov@stealth.net)
**License:** Author-granted freeware ("This scheme is freeware, and is a part of the official KSA OS Scheme Pack"). The scheme reproduces Microsoft's Windows 95 visual style; **Microsoft, Windows, Windows 95, and the Windows logo are trademarks of Microsoft Corporation**.

## About

Windows 95 was Microsoft's 1995 consumer OS, defining the Start-menu-plus-taskbar look that anchored Windows desktop UX for a decade. This scheme retargets the classic Mac OS Appearance chrome to mimic Windows 95.

*"The Windows 95 Scheme for Kaleidoscope 2.x is fully compatible with Kaleidoscope version 2.0 and higher. This scheme is freeware, and is a part of the official KSA OS Scheme Pack."* From the author's ReadMe.

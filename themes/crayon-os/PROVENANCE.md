# Crayon OS

**Author:** Karl von Laudermann (karlvonl@geocities.com)
**Year:** 2000
**License:** Freeware (Karl von Laudermann, ©2000). Author explicitly permits redistribution provided the ReadMe accompanies the bundle and files are unmodified.

## About

Crayon OS renders the Mac OS chrome as if drawn in crayon on manilla paper — a deliberate natural-media riff inspired by Hideaki Kamada's "NotePaper" scheme. Original v1.1 release described in the author's ReadMe as "very demanding... due to its textured nature and realistic look."

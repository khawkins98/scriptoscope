# Apple Platinum (replica) — provenance

A reproduction of the Mac OS 8.5/8.6 **Platinum** appearance, generated by
`scripts/generate-platinum.mjs`. The bundle is built from three layers:

1. **Screenshot-sourced (real pixels)** — the document window and most controls are
   *sliced* from real Mac OS 8 Platinum screenshots stored in `sources/` (owner-
   directed; Platinum is an abandoned, non-enforced appearance, used here for a
   preservation/reproduction project). The slicers map a screenshot region onto the
   cicn the runtime compositor tiles.
2. **Grafted geometry** — control cicns whose shape is correct in the in-repo 1999
   `apple-platinum-2` Kaleidoscope scheme (scrollbar tracks, scroll thumbs, button
   faces, default ring, tabs, grow box) are copied in, recolored to the real
   accent where needed (see below). `scripts/generate-platinum/graft-controls.mjs`.
3. **Procedural fallback** — window types with no screenshot yet (dialog, alert,
   utility, popup, …) are drawn procedurally by `scripts/generate-platinum/
   draw-window.mjs` (the decoded WDEF 125 geometry). This is the *fallback* layer.

Regenerate with `node scripts/generate-platinum.mjs`. The build is deterministic
(`SOURCE_DATE_EPOCH` or a fixed timestamp). `npm test` smoke-checks that the bundle
stays control-complete + valid.

## The Platinum accent

The Mac OS 8.5 default **"Blue" Appearance accent** — ramp dark `#2f1b8e` →
light `#c4cbfc`, sampled from two references that agree (the progress fill in
`progress-dialog-copy-desktop.png` and the scrollbar thumb in the doc-window
shots). Used for the scrollbar thumb (recolored from apple-platinum-2's purple)
and present in the sliced progress fill. The default-button ring is dark (not
accent-colored).

## `sources/` — what's load-bearing

**Used by the generator (load-bearing — do not remove):**

| source | feeds |
|---|---|
| `doc-window-macintosh-hd-active.png` | active document-window cicn (close box, pinstripe, plate, zoom/collapse) |
| `doc-window-macintosh-hd-inactive.png` | inactive document-window cicn (flat bar, no widgets) |
| `general-controls-checks-radios.png` | checkbox (checked/empty), radio (on/off) |
| `progress-dialog-copy-desktop.png` | progress-bar blue fill; disclosure triangles |
| `speech-panel-slider-popups.png` | slider groove + pentagon thumb |

**Staged (kept for future slices — not yet wired):** `doc-window-infinite-hd.png`,
`doc-window-uploads.png` (alternate document-window captures); `alert-stop-cannot-open.png`
(alert window); `dialog-buttons-uncheck-search.png`, `dialog-serialize-serial.png`
(modal dialogs); `list-view-search-sites.png` (list view); `preview-utility-window.png`
(floating utility window); `quicktime-movie-window.png` (media window);
`get-info-snmp-popups.png` (popup menus — note: popups are CSS-drawn, not a themed
cicn control); `folder-icons-system-webpages.png` (Finder folder icons).

## Distribution note

`sources/` contains real Mac OS 8 screenshots, and the generated cicns embed those
pixels. Committed for reproducibility. Revisit before any redistribution — this is a
preservation/reproduction artifact, not a clean-room asset.

## About

Dolphin SOM ("dsom") is a Kaleidoscope scheme distributed via the Mac Themes Garden archive.

No plaintext ReadMe in the archive. The bundle ships an unusual structure (50 AppleScript-derived TEXT entries with whitespace-encoded filenames) alongside the scheme itself; the scheme name "dsom" (Dolphin SOM) is the only consistent identifier.

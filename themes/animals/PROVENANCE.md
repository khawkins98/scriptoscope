## About

Animals is a 1999 Kaleidoscope scheme by Masashi Ichikawa (also the author of Monkey Paradise). The chrome features playful animal motifs across the window types.

From the bundle's HTML ReadMe: *"I wanted to make Animals a fun scheme. Many animals are moving around the window. You will see different pictures even if the window type is the same. See if can find a scene like this one. ... enjoy this scheme!"* The ReadMe carries an *All Rights Reserved* notice; redistribution is implied by the scheme's long-running presence in the Mac Themes Garden archive but is not granted explicitly.

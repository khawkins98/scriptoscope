# Monkey Paradise

**Author:** Masashi Ichikawa (itikawa@rcp.co.jp)
**Year:** 1998
**License:** No explicit license terms in the archive ReadMe ("Welcome Monkey Paradise! I hope you enjoy this color scheme"). Distributed via the period Kaleidoscope community as freeware by convention.

## About

Monkey Paradise is a 1998 Kaleidoscope scheme. The bundle ships rich HTML/GIF documentation including a Christmas variant (the ReadMe references a Santa motif).

*"Welcome Monkey Paradise! I hope you enjoy this color scheme."* — from the bundle's HTML ReadMe (no explicit redistribution clause; included under the community-norm freeware-with-credit convention).

Aaron UI

A web-native runtime for Kaleidoscope themes — and the window manager that hosts them. Loads freeware-licensed Kaleidoscope schemes (chrome, controls, colors) and renders them with CSS, SVG, and JS. Ships mass:werk's "7 Le" as the bundled default.

Phase 1 (WM core) shipped. Phase 4 (theme engine) in flight — schema, extractor, canonical bundles, loadTheme(), and the cinf/ppat renderers have landed; the per-window chrome wiring (#42), bundled-default auto-load (#39), and demo refit (#44) are still open. See GitHub repo and the Phase 4 tracker for what's done and what's next.

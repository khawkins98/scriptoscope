Aaron UI — Mac OS Appearance for the web

Render any classic-Mac Kaleidoscope theme pixel-faithfully in the browser — 1138, BeOS R5, evolution, Apple Platinum 2, more. Pick a scheme from the ribbon to explore: scene + live controls + window playground. Try it on a real page →

Bring your own theme Drop a classic-Mac Kaleidoscope theme file here, or click to choose one — it's decoded and rendered entirely in your browser (nothing is uploaded). Accepts a .sit archive, a .hqx / MacBinary / AppleDouble wrapper, or a raw .rsrc resource fork. New to this? Grab a scheme from the Kaleidoscope archive. Limits: .sitx (StuffIt X) isn't supported yet · a bundle containing several themes shows the largest · this is a preview — the converted theme isn't saved (it's gone on reload).
Window titles render in Charcoal 12, Jeremy Sachs' bitmap clone of Mac OS 8/9 Charcoal (CC BY-SA 3.0). Other UI text uses Virtue, Marty P. Pfeiffer's free Charcoal recreation (Scooter Graphics, 1999) — itself built on Greg Landweber's reverse-engineered Charcoal from Aaron + Kaleidoscope.

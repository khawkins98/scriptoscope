Aaron UI — declarative windows

Every window below is a plain <div data-aaron-window> with content inside it. There is no per-window JavaScript — one bootstrap line scans the page and promotes each tagged element into a faithful classic-Mac window wrapping the live, selectable, reflowing HTML. Drag a title bar to move; drag the bottom-right gripper to resize; click to focus.

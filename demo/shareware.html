Installation

  1. Quit any running copy of SimpleNote 1.x.
  2. Drag the SimpleNote 2.0 application to your Applications folder.
  3. Double-click it. The first launch builds the index (about 30 seconds for 1,000 notes).
  4. Done. Your old 1.x notes are read in place; no migration needed.

System Requirements

  • Mac OS 8.0 or later (System 7.5.5 minimum, with InputSprockets installed)
  • 4 MB free RAM
  • 2 MB disk space + your notes
  • Color QuickDraw (built into Mac OS 7.5+)

SimpleNote 2.0

A tiny note-taking application for Mac OS 8 / 9. One window, no folders, plain text. Indexed for instant search.

Version 2.0 ships with full-text indexing, a System 8.5 Appearance Manager-aware look, and AppleScript support. Backward-compatible with 1.x note files.

SimpleNote is shareware: try it for 30 days; if you keep using it, please send $15 to the address below. Site licenses available for classrooms.

What's New in 2.0

The changes since 1.4:

  • Full-text indexing. Cmd-F finds across every note in milliseconds.
  • Appearance Manager support. SimpleNote now wears whatever Kaleidoscope scheme you have active.
  • AppleScript. Five new verbs: add note, delete note, find, export, count.
  • Drag-and-drop import. Drop a TextEdit clipping on the SimpleNote icon to make a note.
  • Auto-save. Every keystroke. No more lost notes from a frozen system.
  • Dark notes. Cmd-Shift-D toggles a dim-on-dark mode for night use.

Fixed in 2.0

  • Memory leak when scrolling lists over 500 notes (since 1.2)
  • Crash when importing PageMaker text exports
  • Wrong line-ending detection on PC-exported files

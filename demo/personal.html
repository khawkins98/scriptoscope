Hi, I'm Ken.

I make small things on the web. This page is wrapped in Scriptoscope, a library that paints classic Mac OS 8 chrome around any HTML, using the original 1998 Kaleidoscope scheme files as the art source. The chrome you're seeing right now is decoded from a real .sit archive someone uploaded to Info-Mac a quarter-century ago.

Try the theme picker in the top-right; the whole page changes scheme without a reload. Drag any window by its title bar; double-click a title to roll it up. Click the close box on a window to dismiss it (the content stays, as bare HTML in the page).

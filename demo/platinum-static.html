Aaron UI — static Platinum chrome mockup. No JS, no build. Three windows below: active, inactive, and collapsed (windowshade). All CSS lives in this file under --aaron-* custom properties — the same seams a future theme bundle will override.

Welcome

This is what Aaron UI's default Platinum window will look like once Phase 2 ships.

Right now it's a static mockup — the close box doesn't close, the zoom box doesn't zoom, and the growbox is decorative. Phase 1 (the WM core) will make this actually draggable, resizable, and focusable.

Notice the seams: pinstripe titlebar, paper title pill with vertical hairlines, close box on the left, zoom + windowshade on the right, diagonal-stripe growbox bottom-right.

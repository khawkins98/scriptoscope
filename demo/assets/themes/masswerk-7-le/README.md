## Contents

125 PNG files, decoded from 119 cicn (color icon) + 6 ppat (pixel pattern) resources. Filenames preserve the original Kaleidoscope semantic names (e.g., `cicn-n14335-active-document-window.png`).

See `extraction-manifest.json` for the full catalog (resource ID, original name, dimensions, source-format debug info).
